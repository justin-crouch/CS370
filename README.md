## Briefly explain the work that you did on this project: What code were you given? What code did you create yourself?
In this project, I implemented a reinforcement learning algorithm to train an agent to solve a maze. I was initially given code which generated and displayed the maze, as well as code that allowed an agent to move through the maze and score its movement. From this initial code, I implemented the Deep Q-Learning algorithm that has the agent play the game repeatedly. Each action the agent takes is remembered along with how well that action potentially was in solving the maze. After some amount of actions, the agent uses its experience to train its policy network before continuing on taking actions.
